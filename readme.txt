== Description ==
The DS Core plugin optionally includes pre-built stylesheet and script files for building layouts responsively.
This plugin provides the framework required by all divSpot plugins.

= Features =
* Optionally include divSpot stylesheet and script files on the front-end of your website for layout creation and quality of life pre-built CSS.

== Installation ==
1. Log in to your WordPress administrator account and Navigate to 'Plugins -> Add New'.
2. Input "DS Core" in the top right hand side search bar.
3. Click on the "Install Now" button for DS Core.

== Frequently Asked Questions ==
= What stylesheet and script files are included on the front-end of my website? =
DS Core optionally includes the DS Core stylesheet (dsc-style) and script (dsc-script) files on your website. These assets are lightweight.
